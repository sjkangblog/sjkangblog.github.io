블로그/웹사이트에 댓글 기능 추가하기

Posted by Suji Kang on June 30, 2020

오늘 한 작업은 github page로 만든 블로그 게시글 하단에 댓글 기능을 추가하는 작업이다. 직접 html 코드를 이용해서 만든 블로그 또는 웹사이트에 댓글 기능을 어떻게 추가할지 몰라 막막할지도 모른다. 하지만 다행히 Facebook for Developers 페이지를 이용하면 댓글 기능을 아주 쉽게 추가할 수 있다.

Facebook for Developers 페이지: https://developers.facebook.com/docs/plugins/comments#moderation

아래 그림에서 볼 수 있듯이 Facebook for Developers는 댓글 기능을 쉽게 추가할 수 있는 html 코드를 제공하고 있다. 아직 공식 웹사이트에서 한국말로 해당게시글이 번역되지 않은 듯 하다.



간략히 설명하자면,
1. 우선 댓글 기능을 추가하고 싶은 웹페이지/블로그 페이지의 url 주소를 복사한다.
2. 그 다음, "댓글을 남길 URL" 부분에 붙여넣기를 한다. 나는 게시글이 하나씩 올려져 있는 페이지에 댓글 기능을 추가할 것이기 때문에, "게시물 수"를 1로 바꿔주었다.
3. 그 다음, "코드받기" 버튼을 클릭한다.
4. 마지막으로, 받은 두 개의 코드를 연달아 html 코드의 body 태그 부분 중, 댓글 기능을 위치시키고 싶은 부분에 복사 붙여넣기를 하면 된다. 끝! 아주 간단하게 아래 그림처럼 댓글 기능이 추가된 것을 확인할 수 있다.
